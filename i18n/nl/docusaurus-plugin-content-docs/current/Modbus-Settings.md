# Modbus-instellingen

Schakel de Modbus-functie in `Settings` -> `Application` -> `Modbus` in. Dit vereist een herstart.

## Configureerbare opties

| `Option name` | `Description` | `Default value` |
| ----------- | -------------------------------------------------------- | ------------- |
| Max clients Het maximum aantal clients dat verbinding mag maken
| Poort | De poort waarop de Modbus server draait | 502 |
| Time-out: het aantal milliseconden waarna verzoeken worden afgebroken 10000
