# Ondersteuning krijgen

Als u op zoek bent naar EMS-ESP-ondersteuning, een vraag hebt of een suggestie doet, vindt u hier de opties:

- Als je een BBQKees gatewaykaart gebruikt en problemen hebt met het aansluiten, ga dan eerst naar [BBQKees' EMS Gateway Documentation](https://bbqkees-electronics.nl/wiki/).
- Kijk in het gedeelte [Troubleshooting](Troubleshooting.md) voor oplossingen voor veelvoorkomende problemen.
- Probeer de **EMS-ESP Community** op [Discord](https://discord.gg/3J3GgnzpyT). Dit is een goede plek om algemene vragen te stellen en te chatten met andere gebruikers. Je hebt hier meer kans op snelle reacties.
- Doorzoek bestaande **[GitHub issues](https://github.com/emsesp/EMS-ESP32/issues)**, zowel open als gesloten, omdat uw probleem mogelijk al is opgelost.
- **Zoek** in de [GitHub discussions](https://github.com/emsesp/EMS-ESP32/discussions) want misschien vind je een antwoord op je vraag.
- Als laatste redmiddel, **maak** een [Problem Report](https://github.com/emsesp/EMS-ESP32/issues/new?template=bug_report.md) aan op GitHub volgens de onderstaande richtlijnen.

:::tip belangrijk!

    Voordat je een nieuw GitHub probleem aanmaakt, zorg er dan voor dat je de laatste Stable of Development firmware versie gebruikt. Versies ouder dan deze worden niet langer ondersteund, omdat we simpelweg niet de middelen hebben om elke versie te ondersteunen.

    Gebruikers met BBQKees gecertificeerde gateway boards krijgen voorrang bij ondersteuning. Dit is omdat ze garantie hebben en we er zeker van willen zijn dat ze de best mogelijke ondersteuning krijgen. Gebruikers die hun eigen EMS gateway boards hebben gebouwd of een ongevraagd klooncircuit gebruiken, krijgen beperkte ondersteuning omdat we in de meeste gevallen hardware-gerelateerde problemen hebben gevonden.

    Als je een nieuw GitHub issue aanmaakt, zorg er dan voor dat je de volgende informatie verstrekt:

    1. Voeg de EMS-ESP Support Information toe. Dit laat ons zien welke EMS-apparaten je hebt aangesloten en welke firmwareversie je gebruikt. Je kunt de informatie ophalen van de URL [http://&lt;hostname&gt;/api/system/info](http://ems-esp.local/api/system/info) of downloaden door op de knop `Support Information` te klikken op de Help-pagina in de WebUI. Problemen waarbij deze informatie ontbreekt, worden zonder onderzoek gesloten.
    2. Vermeld het probleem dat je ondervindt.
    3. Vermeld de stappen om het probleem te reproduceren.
    4. Vermeld het verwachte gedrag.
    5. Vermeld het werkelijke gedrag.
:::
