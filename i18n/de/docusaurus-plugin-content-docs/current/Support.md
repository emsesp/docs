# Unterstützung erhalten

Wenn Sie auf der Suche nach EMS-ESP-Unterstützung sind, eine Frage haben oder einen Vorschlag machen möchten, haben Sie hier die Möglichkeit dazu:

- Wenn sich die Anfrage auf **Hardware** bezieht, gehen Sie zu [BBQKees' EMS Gateway Documentation](https://bbqkees-electronics.nl/wiki/).
- Die **Support Community** auf [Discord](https://discord.gg/3J3GgnzpyT) ist ein guter Ort, um allgemeine Fragen zu stellen und mit anderen Benutzern zu chatten. Hier haben Sie bessere Chancen, schnelle Antworten zu erhalten.
- **Suchen** Sie in bestehenden offenen und geschlossenen [GitHub issues](https://github.com/emsesp/EMS-ESP32/issues), da Ihr Problem möglicherweise bereits gelöst ist.
- **Suchen** Sie den [GitHub discussions](https://github.com/emsesp/EMS-ESP32/discussions), da Sie vielleicht eine Antwort auf Ihre Frage finden.
- Als letzte Möglichkeit können Sie einen [Problem Report](https://github.com/emsesp/EMS-ESP32/issues/new?template=bug_report.md) auf GitHub erstellen. Siehe unten für die erforderlichen Informationen.

:::tip

    Stellen Sie beim Erstellen eines neuen GitHub-Themas sicher:

    1. fügen Sie die EMS-ESP-Supportinformationen bei. Sie können diese über die URL [http://&lt;hostname&gt;/api/system/info](http://ems-esp.local/api/system/info) oder durch Klicken auf die Schaltfläche `Support Info` auf der Hilfeseite in der WebUI abrufen.
    2. geben Sie die von Ihnen verwendete Firmware-Version an. Sie finden diese in der `Support Info` oder in der WebUI unter der `About` Seite. Es wird dringend empfohlen, dass Sie auf die neueste Entwicklungs-Firmware-Version wechseln, bevor Sie ein Problem erstellen, da das Problem möglicherweise bereits behoben wurde.
    3. geben Sie die von Ihnen verwendete Hardware oder Gateway-Karte an. Sie finden diese in der `Support Info` oder in der WebUI unter der `About` Seite.
    6. geben Sie das Problem an, das bei Ihnen auftritt.
    7. sie die Schritte zur Reproduktion des Problems angeben.
    8. schließen Sie das erwartete Verhalten ein.
    9. sie schließen das tatsächliche Verhalten ein.
:::
