# Domoticz

![logo](/media/logo/domoticz.png)

Domoticz Beta-Versionen ab Dezember 2021 unterstützen das MQTT-Discovery-Protokoll. Da EMS-ESP auch das MQTT-Discovery des Home Assistant vollständig unterstützt, werden Sie automatisch mit allen verfügbaren Entitäten in Domoticz versorgt.

In EMS-ESP v3.3 sind alle Geräte mit Ausnahme der Klimasteuerung als schreibgeschützte Geräte in Domoticz verfügbar. In EMS-ESP v3.4 werden alle Geräte, die vom Benutzer geändert werden können, als einstellbare Geräte in Domoticz verfügbar sein.

Im [BBQKees Wiki](https://bbqkees-electronics.nl/wiki/gateway/domoticz-configuration.html) finden Sie eine Anleitung zur Installation.
