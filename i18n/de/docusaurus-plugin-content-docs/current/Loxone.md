# Loxon

![logo](/media/logo/loxone-logo.png)

EMS-ESP kann in Loxone integriert werden, indem die einfache Restful API verwendet wird, so dass Loxone mit EMS-ESP über HTTP-Anfragen kommunizieren kann. Alternativ kann MQTT verwendet werden, aber Sie müssen einen Raspberry Pi mit Loxberry installieren.

Beide Methoden sind im Detail auf der Seite [@BBQKees Gateway wiki](https://bbqkees-electronics.nl/wiki/gateway/loxone-configuration.html) beschrieben.
