# EMS Telegrams

Diese Seite enthält eine vollständige Liste aller EMS-Telegramme, die von EMS-ESP unterstützt werden.

[Download the full CSV file](/data/dump_telegrams.csv)

Die nachstehende Tabelle wird dynamisch aus der CSV-Datendatei erstellt. Sie können nach Typ-ID oder Name suchen und nach Abrufstatus filtern.

<TelegramsTable />

:::tip

- **Typ-ID**: Der hexadezimale Identifikator für den Telegrammtyp
- **Name**: Der Telegrammname, wie er in EMS-ESP verwendet wird
- **Status**: Zeigt an, ob das Telegramm aktiv von EMS-ESP abgeholt wird

:::
